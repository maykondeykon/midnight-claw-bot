## Lição Aprendida

**Previsão vs Realidade:**

- Realidade: modelo quantizado ocupa 9.6 GB em disco e precisa de ~10 GB RAM para inferência em CPU

**Para futuras instalações de modelos:**
- Verificar `ollama show <modelo> --modelfile` para ver parâmetros reais
- Considerar que CPU inference usa mais RAM que GPU
- Extremis limitado a modelos < 4 GB para inferência confortável
