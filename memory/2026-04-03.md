### 3. Configuração de Exec Approvals
- **Hora:** ~10:06 GMT+1
- **Pergunta:** Como configurar para eu executar scripts sem pedir autorização?
- **Resposta encontrada na documentação:**
  
  **Três controles diferentes:**
  - **Sandbox** (`agents.defaults.sandbox.mode`) — define ONDE as ferramentas rodam (Docker vs host)
  - **Tool Policy** (`tools.allow`/`tools.deny`) — define QUAIS ferramentas estão disponíveis
  - **Elevated** (`tools.elevated.*`) — escape hatch para `exec` rodar no host sem sandbox
  
  **Como evitar aprovações:**
  
  1. **`/elevated full`** — comando que pula exec approvals para a sessão atual
  2. **Config já existe:**
     ```json
     "tools": {
       "elevated": {
         "enabled": true,
         "allowFrom": {
           "telegram": [8593227319]
         }
       }
     }
     ```
  3. **Desabilitar sandbox completamente:**
     ```json
     "agents": {
       "defaults": {
         "sandbox": {
           "mode": "off"
         }
       }
     }
     ```
  
  **Documentação consultada:**
  - `/home/maykon/.npm-global/lib/node_modules/openclaw/docs/gateway/sandbox-vs-tool-policy-vs-elevated.md`
  - `/home/maykon/.npm-global/lib/node_modules/openclaw/docs/channels/telegram.md`
